# BrightStar Assessment Institute — เว็บไซต์ระบบประเมิน (GitHub Pages)

โฟลเดอร์นี้พร้อมอัปโหลดขึ้น GitHub เพื่อเผยแพร่เป็นเว็บไซต์ผ่าน **GitHub Pages** (ฟรี)
เปิด `index.html` เป็นหน้าแรก แล้วลิงก์ไปยังชุดทดสอบและตัวอย่างผลลัพธ์ทั้งหมด

---


## ขั้นตอนการนำขึ้นเว็บ (สำหรับผู้ไม่เคยใช้ GitHub)

### 1. สมัครบัญชี
ไปที่ https://github.com แล้วกด **Sign up** สมัครฟรีด้วยอีเมล

### 2. สร้าง Repository
- กดปุ่ม **+** มุมขวาบน เลือก **New repository**
- ตั้งชื่อ เช่น `brightstar-assessment`
- เลือก **Public** (ถ้าจะเปิดเป็นเว็บสาธารณะ) แล้วกด **Create repository**

### 3. อัปโหลดไฟล์ (ไม่ต้องใช้คำสั่งใด ๆ)
- ในหน้า repository กด **Add file → Upload files**
- ลากไฟล์ **ทั้งหมดในโฟลเดอร์ github_site นี้** (รวม `index.html`) เข้าไป
- กด **Commit changes** ด้านล่าง

### 4. เปิด GitHub Pages
- ไปที่แท็บ **Settings** ของ repository
- เมนูซ้าย เลือก **Pages**
- หัวข้อ **Source** เลือก **Deploy from a branch**
- **Branch** เลือก `main` โฟลเดอร์ `/ (root)` แล้วกด **Save**

### 5. รับลิงก์เว็บ
- รอประมาณ 1–2 นาที แล้วรีเฟรชหน้า Pages
- จะได้ลิงก์รูปแบบ: `https://<ชื่อผู้ใช้>.github.io/<ชื่อ-repo>/`
- เปิดลิงก์นี้จากมือถือหรือคอมเครื่องไหนก็ได้ แชร์ให้ครู/คณะกรรมการได้เลย

## สิ่งที่ GitHub Pages ทำไม่ได้

- ไฟล์ **Word (.docx)** และ **Excel (.xlsx)** จะไม่ "รัน" บนเว็บ เป็นไฟล์ดาวน์โหลดเท่านั้น (เก็บในระบบภายใน)
- เว็บนี้เป็น **เว็บนิ่ง (static)** การกรอกคะแนนแล้วคำนวณสด ๆ ยังต้องทำในไฟล์ Excel ฐานข้อมูลกลางในเครื่อง
- ถ้าต้องการให้ "กรอกคะแนนออนไลน์แล้วบันทึกเข้าระบบ" ต้องพัฒนาเป็นเว็บแอปจริง (มีฐานข้อมูล/หลังบ้าน) ซึ่งเป็นขั้นต่อไปที่ต้องใช้นักพัฒนา
